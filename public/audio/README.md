# Background Music Tracks

This directory contains background music tracks for the AutoTube video pipeline.

## Placeholder Files

The following files are **silent placeholder tracks** that should be replaced with actual royalty-free music:

- `bg-tense.aac` — Tense/dramatic mood background music
- `bg-uplifting.aac` — Uplifting/positive mood background music
- `bg-neutral.aac` — Neutral/ambient mood background music

These placeholders are valid AAC files (44.1 kHz, mono) so the Web Audio API can load them without errors. Replace them with actual royalty-free music tracks of similar format for production use.

## Existing Track

- `ambient-bg.aac` — General ambient background track

## Requirements

All music tracks must be royalty-free and licensed for use in generated video content.
