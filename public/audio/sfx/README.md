# SFX Library for AutoTube

This directory contains sound effects for AutoTube videos.

## Required SFX

Place the following audio files in this directory:

1. `impact-boom.wav` - Deep impact/boom sound
2. `whoosh.wav` - Whoosh transition sound
3. `riser-up.wav` - Rising sound effect
4. `riser-down.wav` - Falling sound effect
5. `impact-slam.wav` - Heavy slam/impact
6. `transition-whoosh.wav` - Transition whoosh
7. `tension-drone.wav` - Tension building drone
8. `ambient-space.wav` - Space/atmospheric ambience

## Supported Formats
- WAV (recommended)
- AAC
- MP3

## Usage
Files should be named descriptively and placed in this directory for the
mixSfxAtBeats function to locate them.
